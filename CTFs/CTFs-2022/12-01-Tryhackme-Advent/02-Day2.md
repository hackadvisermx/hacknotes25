## Common Locations of Log Files

### Windows
- Event Viewer
- Categories: Application - Securty - Setup - System
### Linux
- `/var/log`
- Categories: 
	- Authentication - `auth.log`
	- Package Management - `dpkg.log`
	- Syslog - `syslog`
	- Kernel - `kern.log`
